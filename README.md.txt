# GymFit Database

Projeto de banco de dados da disciplina ADS.  
Sistema de gestão de academia, incluindo alunos, instrutores, planos, pagamentos e aulas.

## Estrutura do projeto

- 01_create_tables.sql — Criação das tabelas
- 02_insert_data.sql — Inserção de dados de exemplo
- 03_select_queries.sql — Consultas SELECT
- 04_updates.sql — Comandos UPDATE
- 05_deletes.sql — Comandos DELETE

## Instruções

1. Executar os scripts SQL na ordem: 01 → 02 → 03 → 04 → 05
2. Garantir que o banco de dados esteja vazio antes de executar os scripts
